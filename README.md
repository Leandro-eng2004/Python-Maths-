# Python-Maths-

This Python project implements numerical methods for mechanical engineering applications. It uses NumPy and Matplotlib to solve mathematical problems and visualize data.

The code includes 3D surface plots for exponential decay functions, numerical ODE solvers using both the Euler method and the 4th Order Runge-Kutta method, and a pi approximation script with convergence error analysis. It also features procedural matrix generation for triangular and binary patterns.

Beyond the calculations, the project outlines practical applications in thermodynamics, fluid mechanics, and structural analysis, such as modeling heat exchangers, pipe network flow, and beam deflection.

Running the script requires Python 3.x with NumPy and Matplotlib. It outputs a series of visual plots for the function surfaces and differential equations, while printing the generated matrices and approximation results to the console.
